## Color Usage Guide

### Primary Actions:
- **Red (#E31E24)**: Main CTAs, important buttons, titles
- **Yellow (#FFD100)**: Secondary CTAs, highlights, accents

### Hover States:
- **Red hover**: #C41A20 (darker)
- **Yellow hover**: #E6BC00 (darker)

### Gradients:
- **Red gradients**: #E31E24 → #C41A20 or #FF6B6B
- **Yellow gradients**: #FFD100 → #FFA500 or #E6BC00

### Backgrounds:
- **Light backgrounds**: red-50, yellow-50
- **Dark backgrounds**: #E31E24 (red)

### Borders & Rings:
- **Primary borders**: #E31E24 (red)
- **Accent borders**: #FFD100 (yellow)
- **Rings**: #FFD100 (yellow)

## Technical Implementation

### CSS Classes:
```css
/* Red Colors */
bg-[#E31E24]      /* Primary red background */
text-[#E31E24]    /* Primary red text */
border-[#E31E24]  /* Primary red border */
hover:bg-[#C41A20] /* Darker red hover */

/* Yellow Colors */
bg-[#FFD100]      /* Primary yellow background */
text-[#FFD100]    /* Primary yellow text */
border-[#FFD100]  /* Primary yellow border */
ring-[#FFD100]    /* Yellow ring */
hover:bg-[#E6BC00] /* Darker yellow hover */

/* Gradients */
from-[#E31E24] to-[#FFD100]  /* Red to Yellow */
from-[#FFD100] to-[#E31E24]  /* Yellow to Red */
```

### Gradient Combinations:
- Login: Red → Yellow → Light Red
- Signup: Yellow → Orange → Red
- Hero: Red-50 → Yellow-50 → White
- CTA: Red → Light Red → Yellow
