### Design System Applied:

#### Colors Used:
- `--primary`: SAR Red for main actions
- `--secondary`: SAR Yellow for accents
- `--foreground`: Text color
- `--background`: Page background
- `--muted`: Subtle backgrounds
- `--border`: Consistent borders
- `--card`: Card backgrounds

#### Utility Classes Used:
- `.glass` - Glass morphism effect
- `.hover-lift` - Lift on hover
- `.shadow-primary` - Brand-colored shadow
- `.bg-gradient-hero` - Red to yellow gradient
- `.transition-all` - Smooth transitions

#### Spacing & Typography:
- Consistent padding: `p-6`, `p-8`, `p-12`
- Consistent gaps: `gap-3`, `gap-4`, `gap-6`
- Rounded corners: `rounded-2xl`, `rounded-3xl`
- Font sizes: `text-4xl`, `text-5xl` for headings
- Line height: `leading-relaxed` for readability
